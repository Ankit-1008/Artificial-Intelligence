# Student Performance Predictor

## Overview

This project predicts whether a student will PASS or FAIL based on study habits and academic performance indicators.

## Machine Learning Concepts

- Classification
- Accuracy
- Precision
- Recall
- Confusion Matrix
- Feature Importance

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Matplotlib
- Seaborn

## Model

Random Forest Classifier

## Dataset

Student Performance Dataset

## How to Run

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Train Model

```bash
python train_model.py
```

### Launch App

```bash
streamlit run app.py
```

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- Confusion Matrix
