# Smart Attendance System using InsightFace

## Overview

Smart Attendance System is an AI-powered attendance management application that uses **InsightFace** for real-time face recognition and automatic attendance marking.

The system captures student face data, trains a recognition model using facial embeddings generated by InsightFace, and automatically records attendance when a registered student is detected through a webcam.

The project eliminates manual attendance processes and demonstrates practical applications of Computer Vision, Deep Learning, and Database Management.

---

## Features

* Student Registration using Webcam
* Face Embedding Extraction with InsightFace
* Face Recognition using Trained Model
* Automatic Attendance Marking
* SQLite Database Storage
* CSV Attendance Backup
* Real-Time Webcam Detection
* Attendance Dashboard
* User-Friendly GUI
* Duplicate Attendance Prevention

## Workflow

### Step 1: Register Student

Capture multiple face images of a student using webcam.

```bash
python register_student.py
```

Images are stored in:

```text
dataset/StudentName/
```

---

### Step 2: Train Model

Generate face embeddings using InsightFace and train the recognition model.

```bash
python train.py
```

Generated files:

```text
models/recognizer.pkl
models/encoder.pkl
```

---

### Step 3: Start Attendance

Launch real-time face recognition.

```bash
python attendance.py
```

When a registered student is detected:

* Name is recognized
* Date and time are recorded
* Attendance is stored in SQLite database
* Attendance is appended to CSV file

---

### Step 4: Open Dashboard

View attendance records and statistics.

```bash
streamlit run dashboard.py
```


